Landing prediction for Falcon 9 rockets

We develop a full Data Science project including data collection, exploratory data analysis, feature engineering and the application of several Machine Learning algorithms for predicting the landing outcome of SpaceX Falcon 9 rockets. We try four different algorithms: Logistic Regression, Support Vector Machine, Decission Tree Classifier and k-Nearest Neighbors. At the end, we evaluate the results and select the best model.
